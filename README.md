# Security Research Notes (Healthcare & Social Care)

## Deutsch

Dieses Repository sammelt meine Notizen, Beobachtungen und Fallstudien
aus dem Bereich IT-Sicherheit, mit Schwerpunkt auf Gesundheits- und
Sozialwesen in Deutschland.

Die Inhalte basieren auf:
- normaler Nutzung von Software im Arbeitsalltag
- passiver Beobachtung (Browser, Entwickler-Tools)
- eigener Analyse und Nachdenken über mögliche Risiken

Es wurden keine Angriffe durchgeführt, keine Exploits entwickelt
und keine produktiven Daten verändert oder abgegriffen.

Der Fokus liegt auf Responsible Disclosure und darauf,
Sicherheitsprobleme früh zu erkennen und sauber zu melden.

---

## English below

This repository contains my security research notes and anonymized case
studies related to healthcare and social care IT systems.

All content is based on:
- regular day-to-day software usage
- passive observation (browser, developer tools)
- personal analysis and threat modeling

No exploitation, no active attacks and no access to production data.

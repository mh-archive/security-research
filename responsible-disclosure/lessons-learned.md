# Lessons Learned

## Deutsch

- Logout bedeutet nicht automatisch das Ende einer Session
- Viele sicherheitsrelevante Hinweise sind im Client sichtbar
- Einzelne Schwächen werden durch Kombination kritisch
- Gesundheits-IT braucht strengere Sicherheitsannahmen
- Rechtliche Grenzen gehören zur technischen Arbeit dazu

---

## English below

- Logout does not automatically terminate a session
- Many security issues are visible on the client side
- Combined weaknesses are more dangerous than single issues
- Healthcare IT requires stricter security assumptions
- Legal awareness is part of security work
